Reisenpreismethode

Ein kleines Java-Projekt zur Berechnung von Reisekosten.
Das Programm berechnet automatisch den gesamten Reisepreis für Erwachsene und Kinder inklusive Rabatten, Mehrwertsteuer und Kurtaxe.

Funktionen
Berechnung des Grundpreises
Rabatt für Kinder
Gruppenrabatt
Online-Rabatt
Berechnung der Mehrwertsteuer
Berechnung der Kurtaxe
Ausgabe des Endpreises
Verwendete Variablen
Anzahl der Erwachsenen
Preis pro Erwachsenen
Anzahl der Kinder
Preis pro Kind
Anzahl der Nächte
Kurtaxe pro Nacht
