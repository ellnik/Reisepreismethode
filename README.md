# Reisenpreismethode

Ein kleines Java-Projekt zur Berechnung von Reisekosten.

Das Programm berechnet automatisch den gesamten Reisepreis für Erwachsene und Kinder unter Berücksichtigung von Rabatten, Mehrwertsteuer und Kurtaxe.

---

## Funktionen

- ✅ Berechnung des Grundpreises
- ✅ Rabatt für Kinder
- ✅ Gruppenrabatt
- ✅ Online-Rabatt
- ✅ Berechnung der Mehrwertsteuer
- ✅ Berechnung der Kurtaxe
- ✅ Ausgabe des Endpreises

---

## Verwendete Variablen

- Anzahl der Erwachsenen
- Preis pro Erwachsenen
- Anzahl der Kinder
- Preis pro Kind
- Anzahl der Übernachtungen
- Kurtaxe pro Nacht

---

## Berechnungen

Das Programm berechnet automatisch:

- den Grundpreis der Reise
- alle Rabatte
- die Mehrwertsteuer
- die Kurtaxe
- den Endpreis

### Beispielausgabe

```text
Zu zahlen: 3377,00 €
```

---

## Besondere Methode

```java
static double prozentbetrag(double grundbetrag, double prozentzahl)
```

Diese Methode berechnet den Prozentbetrag eines Grundwertes und wird zur Ermittlung von Rabatten oder Zuschlägen verwendet.

---

## Verwendete Technologien

- Java
- Eclipse IDE

---

## Projekt starten

1. Projekt in Eclipse öffnen.
2. `Reisenpreismethode.java` starten.
3. Die Eingabewerte im Code anpassen.
4. Das Programm ausführen und den berechneten Reisepreis in der Konsole anzeigen lassen.
